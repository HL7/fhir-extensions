

All extensions in this list are defined in this specification and have a base URI of 
```http://hl7.org/fhir/StructureDefinition/```. 
This page does not list inter-version conversion extensions - see guidance [here]({{site.data.fhir.path}}versions.html#extensions).
Additional extensions can be registered on the HL7 FHIR registry at [http://hl7.org/fhir/registry](http://hl7.org/fhir/registry).
See the [end of this page](#legend) for an explanation of this table.

**Legend**

This table contains the following columns:

* `Identity`: The tail of the URL (the bit that follows http://hl7.org/fhir)
* `Card.`: The [defined cardinality](http://hl7.org/fhir/R5/defining-extensions.html#cardinality) of the Extension, and the letter **M** if the extension is a modifier extension
* `Type`: The type of the Extension value (or '(Choice)', '(Complex)')
* `Context`: Where the extension is [allowed to be used](http://hl7.org/fhir/R5/defining-extensions.html#context)
* `WG`: The HL7 Work Group that is responsible for the definition of the extension 
* `Status`: A summary of the [status](http://hl7.org/fhir/R5/versions.html#std-process) and maturity of the extension
* `Î” v5.2.0`: Information about whether the definition of the extension has changed since the last publication
